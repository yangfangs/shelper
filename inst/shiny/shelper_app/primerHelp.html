Shelper Tool Help Document

How to Input Data

This section of the help document provides a guide on how to input data into the ADPV Tool.

  1. Select the Genome Version you would like to use for your analysis. You can choose between:
    • hg19 (GRCh37)
    • hg38 (GRCh38)
  2. Enter the Genome Location in the input field provided. The format should match the example given, such as '13-32914859-G-GA'.
  3. You can load example data to see how the tool works by clicking the 'Load Example Data' button.
  4. Once you have entered the required information, click 'Submit' to process your data.

Understanding the Output

The output of the Shelper Tool is presented in a tabulated format with the following columns:

  • Forward/Reverse: Indicates whether the primer is a forward or reverse primer.
  • Validate location: Shows the genome location that was input for validation.
  • Sequence (5'-3'): Displays the nucleotide sequence of the primer.
  • Start: Denotes the starting position of the primer on the genome.
  • TM: Shows the melting temperature of the primer, important for PCR conditions.
  • %GC: Indicates the percentage of guanine and cytosine in the primer sequence, which affects the primer's stability.
  • Self complementarity: Displays the primer's potential for forming secondary structures with itself, which can affect the efficiency of the PCR.
  • Self 3' complementarity: Similar to self complementarity but specifically at the 3' end of the primer.
  • Product length: Shows the expected size of the PCR product generated with the primer pair.

Each primer pair is listed with its respective details. Ensure that the primers chosen have optimal TM, %GC, and complementarity scores for efficient PCR amplification.
